# Minecraft Server Infrastructure (Public Portfolio Version)

Projekt przedstawia czystą, bezpieczną i profesjonalną strukturę serwera Minecraft
w formie przejrzystego repozytorium. Repozytorium nie zawiera żadnych wrażliwych danych,
haseł, tokenów, adresów IP ani plików prywatnych.

## Zawartość katalogów

- **scripts/**  
  Skrypty automatyzujące pracę administratora (backup, status, update, logi).
  Wszystkie skrypty będą przygotowane tak, aby działały na dowolnym serwerze
  bez ujawniania informacji prywatnych.

- **examples/**  
  Przykładowe pliki konfiguracyjne (server.properties, whitelist.json, ewentualnie
  przykłady systemd), oczyszczone z danych użytkownika.

- **docs/**  
  Dokumentacja techniczna opisująca:
  - architekturę serwera
  - wymagania sprzętowe
  - polityki bezpieczeństwa
  - procedury backupu
  - procedury aktualizacji

## Cel projektu

Celem repozytorium jest przedstawienie profesjonalnych umiejętności DevOps
w kontekście zarządzania serwerem Minecraft, w sposób w pełni bezpieczny i publiczny.

Repozytorium ma służyć jako:
- portfolio dla rekrutera,
- dokumentacja własnej infrastruktury,
- baza dobrych praktyk administracyjnych.

## Uwaga

Repozytorium NIE zawiera:
- prawdziwych danych serwera,
- adresów IP,
- tokenów, kluczy SSH ani haseł,
- prywatnych pluginów, modów ani licencji.

Wszystkie dane są oparte na przykładach technicznych.
